# 📈 港股IPO内容生成引擎

一个基于AI的内容生成系统，用于将港股IPO公开信息转化为小红书风格的内容表达。

---

## 🚀 Demo（在线体验）

👉 https://你的streamlit网址

---

## 🧠 项目结构

IPO信息 → 数据提取 → 多风格生成 → 内容输出

支持三种风格：

* 🧨 情绪交易派（conflict）
* 🧊 冷静分析派（rational）
* 🎭 叙事观察派（story）

---

## ⚙️ 技术栈

* Python + Streamlit
* OpenAI API（或兼容接口）
* SerpAPI

---

## 💡 核心能力

* 自动抓取IPO相关信息
* 结构化提取关键数据（ipo_card）
* 多风格内容生成（避免重复）
* 去AI味优化

---

## 🎯 使用场景

* 港股打新内容创作
* 投研表达辅助
* 自媒体内容生成
* Demo / 产品展示
